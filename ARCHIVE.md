This branch preserves the complete repository state of the clean-room
recursive spatial puzzle study before the project direction changed. It is a
research and prototype archive, not a finished game, release, or official
Patrick's Parabox implementation.

independent QA was intentionally stopped when the user chose to archive the
project. The archive therefore does not label R2I as integrated, accepted, or
release-ready.

## Completion boundary

The overall target remained below 10% complete. In particular, the following
work was not completed:

- V2 detached-void/cropped-parent composition and final material system;
- V3 retained recursive scene graph and performance/heap acceptance;
- V4 responsive input, DPR cap, reduced motion, accessibility, and capture
  automation;
- production level schema, serialization, validator, loader, solver, campaign,
  or accepted showcase level;
- cyclic recursion, full content parity, copied levels, official assets, audio,
  branding, or text.

## Resume boundary

If this study is ever resumed, start by reading `AGENTS.md`, `CURRENT_TASK.md`,
`DESIGN.md`, the accepted rules contracts, and all workstream logs. Re-run an
independent review of the exact R2I chain before treating it as accepted. Do
not infer completion from historical Stage labels or from the existence of
screenshots.

The repository `main` branch is intentionally returned to a minimal neutral
starting point for a different future game.
